# 🚀 Striver SDE Sheet Challenge

This repository contains my solutions, approaches, and learnings from the Striver SDE Sheet Challenge.

The objective is to strengthen problem-solving skills, improve consistency, and build a structured collection of interview-oriented Data Structures and Algorithms solutions.

## 💻 Language

* Java

## 🎯 Goals

* Solve problems consistently.
* Understand patterns instead of memorizing solutions.
* Document approaches and key learnings.
* Improve coding and interview preparation skills.

## 📂 Repository Structure

```text
Day-01/
Day-02/
Day-03/
...
```

Each day's folder contains:

* Solution code
* Problem-specific notes
* Complexity analysis

## 📈 Topics Covered

* Arrays
* Strings
* Linked Lists
* Stacks & Queues
* Recursion & Backtracking
* Binary Search
* Trees
* Graphs
* Dynamic Programming
* Greedy Algorithms
* And more...

## 🌱 Learning Philosophy

The focus is not just on solving problems, but on understanding the intuition, identifying patterns, and developing strong problem-solving skills that can be applied in interviews and real-world scenarios.
